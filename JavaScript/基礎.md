# DOM
DOM(Document Object Model)

*HTMLの書き換えの例

```html
<html>
<body>
    <h1 id="title">天気予報</h1>
    <p id="content">今日の天気</p>
</body>
</html>
```

* ページの内容を書き換える
```JavaScript
document.getElementById('content').firstChild.nodeValue = '明日の天気';
```

* ブラウザに保存されているcookieを表示する
```JavaScript
document.cookie
```

* 警告を表示する
```JavaScript
alert()
```

* 別のページにリダイレクトする
```JavaScript
location.href = "URL"
```

* 情報を攻撃者に送信する
→document.cookieとLocation.hrefの組み合わせ
